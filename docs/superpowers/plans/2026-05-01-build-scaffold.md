# Chimera Build Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a reproducible local Bun/TypeScript scaffold around the recovered `src/` tree so dependency, typecheck, and build blockers are visible and actionable.

**Architecture:** Keep the original source tree untouched except for small missing type shims. Put build tooling at the repo root, generated diagnostics under `scripts/`, and private/native module placeholders under `stubs/` so we can replace them with real packages later.

**Tech Stack:** Bun, TypeScript, local file dependencies for unavailable private packages, shell scripts for diagnostics.
