# Chimera Local Scope

## Product Contract

Chimera is a local coding agent CLI/TUI backed by ChatGPT/Codex OAuth and real OpenAI model IDs.

## Allowed Local Surfaces

- CLI/TUI.
- Local transcripts and resume.
- Local fork, compact, and session management.
- Local tools.
- Local agents and subagents.
- Local MCP, LSP, plugins, hooks, and settings.
- Local browser/CDP automation.
- Optional native desktop adapters.
- Legacy import compatibility for existing Claude Code configuration files.
- `CLAUDE.md` project instruction compatibility alongside `AGENTS.md`.

## Deleted Cloud Surfaces

- Claude cloud assistant attach/viewer mode.
- Remote bridge and peer sessions.
- Cloud daemon, environment runner, and self-hosted runner.
- Cloud notifications and subscribe-PR tools.
- Cloud direct connect.
- Anthropic model aliases and provider selection.
- Anthropic private docs and endpoints.
